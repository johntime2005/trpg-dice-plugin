# 重构更新日志

## v2.0.0 - AI 增强版

### 新增功能

#### 智能剧情引擎
- 动态场景管理和氛围控制
- 自动紧张度调整
- 线索追踪系统

#### NPC 管理系统
- NPC 人格和记忆模拟
- 关系网络追踪
- 动态对话生成支持

#### 增强提示词系统
- 5 层提示词注入架构
- 情境感知的叙事风格调整
- 即兴创作指导
- 智能检定哲学

#### 情境感知检定
- 动态难度调整
- 失败推进剧情机制

### 新增沙盒方法

```python
set_scene(scene_name, mood)  # 设置场景
set_tension(level)  # 调整紧张度
add_clue(clue)  # 记录线索
create_npc(...)  # 创建 NPC
npc_remember(npc_id, memory)  # NPC 记忆
update_npc_relationship(...)  # 更新关系
```
