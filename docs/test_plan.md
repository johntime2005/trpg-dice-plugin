# TRPG 骰子插件 - 系统测试计划

## 一、基础掷骰测试

### 1.1 基本表达式
| 测试用例 | 指令 | 预期结果 |
|---------|------|---------|
| 单骰 | `r d20` | 掷1个d20，结果1-20 |
| 多骰 | `r 3d6` | 掷3个d6，显示每个骰子结果和总和 |
| 带修正值 | `r 2d10+5` | 掷2个d10+5 |
| 复合表达式 | `r d4+d6+d8` | 多种骰子组合 |
| 减法修正 | `r 3d6-2` | 掷3个d6-2 |
| 大数量骰子 | `r 10d6` | 掷10个d6 |

### 1.2 边界条件
| 测试用例 | 指令 | 预期结果 |
|---------|------|---------|
| 最大骰子数 | `r 100d6` | 接近上限不崩溃 |
| 超限骰子数 | `r 101d6` | 返回错误提示 |
| 最大面数 | `r d1000` | 正常工作 |
| 超限面数 | `r d1001` | 返回错误提示 |
| d1 | `r d1` | 结果总是1 |

### 1.3 隐藏掷骰
| 测试用例 | 指令 | 预期结果 |
|---------|------|---------|
| 隐藏掷骰 | `rh d20+5` | 只显示最终结果 |
