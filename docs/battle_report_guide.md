# 📊 TRPG战报系统

## 核心功能

### 自动记录
自动记录所有游戏活动：投骰(`r`)、检定(`ra`)、行动(`me`)、大成功/大失败

**智能启动**: 首次投骰/检定/行动时自动开始记录，无需手动命令  
**历史记忆**: 自动加载上次战报给AI，实现剧情连续性

### 玩家评分
自动计算5星级评分，基于：参与度、技能成功率、角色扮演、精彩时刻  
评级：⭐⭐⭐⭐⭐传奇 / ⭐⭐⭐⭐优秀 / ⭐⭐⭐良好 / ⭐⭐合格 / ⭐需努力

## 使用方法

### 自动模式（推荐）
```bash
# 直接开始游戏，自动启动记录
r 3d6+2
ra 侦察
me 观察房间

# 结束时生成战报
session end
```

### 手动模式（可选）
```bash
# 自定义会话名称
session start 深海古城探险

# 记录关键事件
session event 发现神秘入口

# 生成战报
session end
```

## 战报内容
- 会话信息（名称、时长）
- 玩家评分和统计
- 游戏数据汇总
- 关键事件回顾
- 精彩时刻
- Markdown文档导出

## AI沙盒方法
- `start_session_recording(session_name)` - 开始记录
- `add_session_event(description, event_type)` - 记录事件
- `generate_session_report()` - 生成战报
